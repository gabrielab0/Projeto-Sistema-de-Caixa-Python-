# Projeto-Sistema-de-Caixa-Python-
Projeto desenvolvido em Python com foco em lógica de programação e modelagem de regras de negócio.
O sistema simula um caixa de loja, realizando o cálculo do valor total da compra a partir de produtos com preço e quantidade, aplicação de regras de desconto e parcelamento conforme a forma de pagamento, uso de loops para processamento dos itens e funções para organização e reutilização do código.
